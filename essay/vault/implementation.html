Policy

讀取現有的政策

$ vault read sys/policy

Key         Value
---         -----
keys        [default root]
policies    [default root]

建立政策

$ vault policy write my-policy policy.hcl

policy.hcl 檔：

path "secret/*" {
  capabilities = ["create", "read", "update", "delete", "list"]
}

path "secret/hello" {
  capabilities = ["deny"]
}

path "secret/restricted" {
  capabilities = ["create"]
  allowed_parameters = {
    "foo" = []
    "bar" = ["zip", "zap"]
  }
}
