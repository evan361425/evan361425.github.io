# 漫畫推薦

一直想著要把自己喜歡的漫畫整理起來，並寫上一些簡單的心得，
未來的自己如果看到這些心得將會是個很有趣的狀況吧 😂。
當然，對於評論這些心血結晶我絕對是沒有資格的，
能讓我厚著臉皮寫下這些評論的只能是因為寫這些評論時是真心而嚴肅的，
如果哪一天我意識到自己正在胡謅鬼扯時，希望有人或自己能及時發現並提醒自己該好好冷靜下來。

這份推薦清單可能不會有那些「經典」之作，例如怪醫黑傑克、多拉A夢等等。
並不是因為這些作品不好看，而是基於小時候都是透過卡通（動畫）來認識這些作品，
讓我對於要用漫畫的角度來看這些作品時竟有些彆扭，也許他們在我心中並不歸類於漫畫吧。

在接下來的料表中，第一層會列出作者第二層再列出作品，如果作品是以原作（通常是小說）為基礎的，
就會在作品名後面加上原作作者，例如「井上雄彥—浪人劍客 + 吉川英治」。
第三層代表其作品是上一層作品的延伸作品，例如「請叫我英雄—公式合集」
